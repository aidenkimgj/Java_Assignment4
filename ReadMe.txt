Project Title : Flight reservation program

This program is that Users can find and reserve flight information through resource files.
Resource files have airport information, flights information and reservation information.

The date : July 10, 2020

The author : Sangmean Hong, Aiden(Gookjin) Kim

Users can run this program by running a file with a "jar" extension on the command prompt. 
Users should type "java -jar filename.jar" on the command prompt.

When the software program show up on the screen, user can find and reserve and update flight information.
